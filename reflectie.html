Persoonlijke situatie

Tijdens het maken van deze portfolio heb ik geprobeerd om een ZIP-bestand van het project te maken, maar dat is mij niet gelukt. Ook ben ik een paar dagen in het ziekenhuis geweest voor controles in verband met een operatie en mijn gezondheid (bloeddruk). Daardoor heb ik minder tijd gehad om de website nog mooier of uitgebreider te maken.

Ondanks dat ben ik wel trots op wat ik heb kunnen maken en heb ik veel geleerd tijdens dit project. In de toekomst wil ik de website verder verbeteren en uitbreiden.

Wat ik heb geleerd

  • HTML structuur verbeteren
  • CSS dark/light mode maken
  • Bootstrap gebruiken
  • Projecten organiseren

Verbeterpunten

  • Meer animaties toevoegen
  • Betere UX design
  • Projecten uitbreiden
