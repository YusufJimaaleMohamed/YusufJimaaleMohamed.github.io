Over deze website

Deze portfolio heb ik gemaakt voor mijn opleiding (MBO-4 Software Development). Het doel van deze website is om mijn projecten te laten zien en mijn groei als developer te laten zien.

Ik heb gewerkt met HTML, CSS, JavaScript en Bootstrap om een moderne en responsive website te bouwen die op alle apparaten goed werkt.

Inspiratie

Ik heb inspiratie gehaald uit moderne developer portfolio’s op GitHub en andere websites. Vooral minimalistische designs, dark mode en duidelijke project cards vond ik interessant en heb ik geprobeerd zelf toe te passen.

Technieken

  • HTML5 (gestructureerde pagina’s)
  • CSS (custom variables + dark/light mode)
  • JavaScript (theme switch en interactie)
  • Bootstrap (layout en responsive design)

Wat ik wil verbeteren

  • Meer animaties en interactie toevoegen
  • UI/UX design verder verbeteren
  • Meer en grotere projecten toevoegen
  • De website nog professioneler laten ogen
