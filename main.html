Over deze website

Deze website heb ik gemaakt voor mijn portfolio in periode 4. Ik wilde iets dat duidelijk is, rustig oogt en laat zien wat ik kan. Ik heb gewerkt met HTML, CSS, Bootstrap, JavaScript en AOS.

Inspiratie

Ik heb gekeken naar moderne portfolio’s en simpele websites. Vooral AOS vond ik handig voor animaties.

Kleurenschema

De website gebruikt een gradient die verandert met dark/light mode. Dit maakt het rustig en modern.
